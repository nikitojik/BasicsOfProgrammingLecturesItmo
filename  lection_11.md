# Перегрузка операторов
1. **не должна противоречить здравой логике**
2. как член класса или как глобальная функция
3. [],(),->,= всегда члены класса
4. ввод(>>) и вывод(<<) всегда глобальные функции
5. операторы ::,.*,.,?: перегружать нельзя
6. новые операторы делать нельзя (на уровне компиляции зафиксировано)

## ключевое слово **friend**
- если внешняя функция является friend нашего класса, то она получает приватные поля нашего класса

```cpp
friend void pop(int arr[],int); // внутри класса

void pop(int arr[],int){ // внешняя функция 
    //реализация
}
```

## функциональные объекты или функтор
объект, который ведет себя как функция - отличается от функции, потому что есть состояние
```cpp
class CMult{
    public:
        explicit CMult(int mult)
            :mult_(mult)
        {}

        int operator()(int value){
            return mult_ * value;
        }
    private:
        int mult_;
};

int main(){
    CMult twice{2};
    std::cout << twice2(3)<< std::endl;
}

```
