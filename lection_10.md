# ООП
- абстракция (бинарное дерево, куча, Stack)
- наследование
- полиморфизм
- инкапсуляция 

## Инвариант - выражение, определяющее непротеворечивое внутренее состояние объекта.(идея при разработке)

## class - комплексный тип данных...

## абстракция - придание объекту характеристик, которые четко поределяют его концептуальные границы, 
отличая от всех других объектов. Основная система состоит в том, чтобы отделить способ использования составных данных
от деталей.

## инкапсуляция - связывает данные и методы работы с этими данными, (сокрытие внутренних деталей реализации 
от внешнего мира). Класс объединяет данные и методы работы с этими данными. Данные мы прячем за реализацией

## конструктор
- спец. не статический метод, используемый для инициализации объекта(обеспечивает инвариант)
- название метода совпадает с именем класса
- **виды конструкторов**
    - default constructor
    - parameterized constructor
    - copy constructor (создание нового объекта, копируя из сущ. объекта)
    - move constructor
