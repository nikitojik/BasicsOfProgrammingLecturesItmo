## Перегрузка функций
функции с одинаковыми названиями, но разными сигнатурами, нужно чтобы оперировать с разными типами данных

компилятор ищет подходящую функцию, если вариантов больше двух, то выдается ошибка компиляции. 
Для избежания стоит использовать явное преобразование.
нельзя, чтобы функции возвращали разные значения

## namespace
- предотвращает конфликт имен
- могут состоять из нескольких блоков
- упрощают читабельность кода
- unnamed namespace
- namespace alias (есть такое-то имя, которое интерпритирует какой-то namespace)

в разных библиотеках могут встречаться функции с одинаковыми названиями и одинаковыми сигнатурами

namespace - это пространство имен которые объединяют логически связанные блоки кода 

using namespace Foo; // принято так не делать для читабельностьи кода
int main(){
    f(); - компилятор смотрит и в namespace Foo поэтому можем явно не указывать namespace
    Boo::f();
}
