# Reference, initialization, function overloading namespaces

#Reference (ссылки)
- "Псевдоним" для уже существующего объекта
- ссылка обязана быть инициализирована 
- не занимает доп. память
- нельзя сделать указатель на ссылку
- продлеает жизнь временным переменным
- могут быть const(можно сделать на неконстантную переменную)

пример:
int main(){
    int i = 10;
    int& j = i; // int& ccылка на int
    j = 20;
    std::cout << i << std::endl;
    std :: cout << &i << ' ' << &j <<  std::endl;
    return 0;
}
